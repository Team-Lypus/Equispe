## :toolbox: Getting Started

### :bangbang: Prerequisites

- Install docker<a href="https://www.docker.com/"> Here</a>


### :running: Run Locally

Clone the project

```bash
https://github.com/Team-Lypus/Equispe
```
Go to the project directory
```bash
cd Equispe
```
Start docker compose
```bash
docker-compose up -d
```
